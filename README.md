# Privacy Spotlight

Chrome extension che nội dung website để hạn chế người phía sau đọc màn hình. Mỗi website phải được người dùng bật riêng trong popup; lựa chọn được lưu theo domain.

## Cài đặt thủ công

1. Mở `chrome://extensions`.
2. Bật **Developer mode**.
3. Chọn **Load unpacked**.
4. Chọn thư mục dự án này.
5. Mở website cần bảo vệ, bấm biểu tượng extension và bật **Bật trên website này**.
6. Tải lại các tab đã mở trước khi cài extension.

Mặc định, một hội thoại trong sidebar sẽ hiện rõ khi hover. Tùy chọn yêu cầu giữ `Alt` nằm trong trang cài đặt của extension.
